# Pinia
全局状态管理工具

[官方文档](https://pinia.vuejs.org/)
[官方中文文档](https://pinia.web3doc.top/)

## 特点
* 完整的ts支持
* 足够轻量，压缩后体积只有1KB左右
* 去除mutations，只有state,getters,actions
* actions 支持同步和异步
* 代码扁平化没有模块嵌套，只有store概念，store之间可以自由使用，每一个store都是独立的
* 无需手动添加store，store一旦创建便会自动添加
* 支持Vue3 和 Vue2

## 安装
`npm i pinia -S`
